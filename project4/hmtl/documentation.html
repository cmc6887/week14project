Landing Page

For this project, I used Adobe Illustrator, Adobe Photoshop, and Brackets. I used my prior Photoshop experience to aid my websites looks. When working on this project I learned quite a few things. I first learned how to use Adobe Illustrator, in order to make the SVG graphics used on my pages. I also learned a lot about JavaScript. I learned one way how to use a timer. I learned how to and the uses of running multiple of the same function at the same time. I learned how to move divs to different locations based on the screen resolution, and or the window size. I also learned how to repeat functions through a timer. I even learned some more funky css transitions such as a blur effect. For this project I went above and beyond on the both the layout and functionalities for these pages. For each page I made very different mini games all with a central theme but all very different and they even have a practical use! I went above and beyond for each pages functionalities, adding start buttons to two pages and adding high scores to two pages as well. I went above and beyond learning new features never talked about in this class for JavaScript, such as the ones I learned from this project. I also went above and beyond making the backgrounds for each page, and making the menu icons for the landing page. Those pictures took me quite a while, especially the silhouette ones based off real counterstrike maps. There were two main resources I used throughout this project.
